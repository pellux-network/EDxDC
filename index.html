EDxDC
Real-time Elite Dangerous journal data on your X52 Pro MFD.
Instantly see system, planet, and cargo info as you play.
Note: Only works with the X52 Pro (not the regular X52 HOTAS).
Note: EDxDC is currently in beta. Please report bugs on GitHub Issues.
Station & Planet Details
See station, fleet carrier, and planet info at a glance.
Real-Time Cargo Tracking
Monitor your cargo hold and materials instantly.
Surface Gravity & Scan Value
Get surface gravity and scan value indicators for planets.
Context-Aware Pages
Pages switch automatically based on your current activity.
Manual EDSM Data Refresh
Update exploration data on demand to ensure you have the latest system information.
Scoopability & Route Info
See scoopable stars and jump route information.

Quick Start

  1. Install the official Logitech X52 Pro drivers
  2. Download the latest EDxDC installer from GitHub and launch it
  3. After installation, the app is ready to run.
    If needed, the config file is located at: %APPDATA%\EDxDC\main.conf
Note: For detailed setup, see the Getting Started guide.
